Adding additional files

If the functionality you're adding doesn't happen to fit into one of the existing files (app/helpers.php, app/setup.php, app/filters.php, app/admin.php) then it might best belong in a functionality plugin outside of the theme.

If the functionality you're adding is theme specific, you can create a new file app/<name>.php file and add it to the $sage_includes array in functions.php. Here's an example of adding app/woocommerce.php:

// @ functions.php
$sage_includes = [
    'app/helpers.php',     // Helper functions
    'app/setup.php',       // Theme setup
    'app/filters.php',     // Filters
    'app/admin.php',       // Admin
    'app/woocommerce.php', // WooCommerce functions
];

And a couple things that you might find in that file:

// @ app/woocommerce.php
/**
 * Use 'product-lg' image size on WooCommerce product archive
 */
add_filter('single_product_archive_thumbnail_size', function () {
    return 'product-lg';
});

/**
 * Change Gravatar size for reviews
 */
add_filter('woocommerce_review_gravatar_size', function () {
    return 120;
});
