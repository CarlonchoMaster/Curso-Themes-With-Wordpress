Bootstrap

Bootstrap is included as the default option in Sage. The primary choice for using Bootstrap is that it's the most popular front-end framework.

Bootstrap options

Bootstrap's global options can be defined in _variables.scss:

$spacer:              1rem;  // Default spacer value
$enable-rounded:      true;  // Enable rounded corners on components
$enable-shadows:      false; // Enable box-shadow on components
$enable-gradients:    false; // Enable gradients on components
$enable-transitions:  true;  // Enable transitions on components
$enable-grid-classes: true;  // Enables grid classes  (.container, .row, .col-md-1, etc.)

Bootstrap variables

There are a lot of variables available in Bootstrap to allow for customization without having to override styles.

Copy and paste relevant lines from Bootstrap's _variables.scss into Sage's _variables.scss file without the !default flag (or copy/paste all lines).

To customize the breakpoints for Bootstrap's grid, you would copy the following into _variables.scss and make your changes:

$grid-breakpoints: (
  // Extra small screen / phone
  xs: 0,
  // Small screen / phone
  sm: 576px,
  // Medium screen / tablet
  md: 768px,
  // Large screen / desktop
  lg: 992px,
  // Extra large screen / wide desktop
  xl: 1200px
);

$container-max-widths: (
  sm: 540px,
  md: 720px,
  lg: 960px,
  xl: 1140px
);

Brand variables

Sage's _variables.scss file is also the place for all of your brand related variables. Here's a quick example of using Bootstrap's variables to customize the look and feel of a theme:

// Colors
$green: #27ae60;

$theme-colors: (
  primary: $green
);

$headings-color:     theme-color('primary');
$table-border-color: theme-color('primary');
$card-border-color:  theme-color('primary');

Bootstrap utility classes

Bootstrap includes many utility classes for applying single rules or universal patterns, such as .clearfix, .text-primary, .text-uppercase, .align-middle and more.

Lots of the utility classes are responsive and use the same breakpoints as the grid system, for instance:

<p class="text-center text-md-left">Roots helps you build better WordPress sites faster.</p>
  • .text-center — Align text to the center on all viewport sizes
  • .text-md-left — Align text to the left on medium or larger viewport sizes

The flexbox grid utility classes are great for aligning things:

  • .align-items-center
  • .align-items-stretch
  • .align-items-end

You can view all of the utilities on Bootstrap's docs or directly in the source.

The spacing utilities are particularly helpful for adding margin or padding to any sides of an element.

Bootstrap mixins

Bootstrap's mixins are helpful to use in your own stylesheets when you've defined your own additional brand colors to use in the theme.

// resources/assets/styles/common/_variables.scss
$green:     #2ecc40;
$red:       #ff4136;
$blue:      #0074d9;

// resources/assets/styles/common/_global.scss
@include bg-variant('.bg-green', $green);
@include bg-variant('.bg-red', $red);
@include bg-variant('.bg-blue', $blue);

// resources/assets/styles/common/_buttons.scss
.btn-secondary {
  @include button-variant(#fff, $green, $green);
}

.btn-red {
  @include button-variant(#fff, $red, $red);
}

.btn-blue {
  @include button-variant(#fff, $blue, $blue);
}

Responsive typography

Since Bootstrap 4 uses rem sizing, setting up basic responive typography is nice and easy.

html {
  font-size: 1rem;
}

@include media-breakpoint-up(md) {
  html {
    font-size: 1.2rem;
  }
}

@include media-breakpoint-up(xl) {
  html {
    font-size: 1.4rem;
  }
}

You can also extend these rules for specifically tweaking sizes on other typography elements, such as headings:

html {
  font-size: 1rem;
}

h1 {
  font-size: 1.4rem;
}

@include media-breakpoint-up(md) {
  html {
    font-size: 1.2rem;
  }

  h1 {
    font-size: 1.8rem;
  }
}

Follow twbs/bootstrap#23053 for the responsive typography updates coming to Bootstrap 4.1.
