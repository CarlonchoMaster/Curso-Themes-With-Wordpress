3rd party packages

To add third-party JavaScript or jQuery plugins to your theme, it’s best to do so with npm, the package manager for JavaScript, whenever possible. Most jQuery plugins and JS libraries are available as npm packages.

Here's how to add Slick Carousel and have the CSS and JS included in your theme:

  1. First find the package name by either looking at the name defined in package.json, or by searching on https://www.npmjs.com/. Once you've found the name of your package, install it from your theme directory:

     # @ themes/your-theme-name/
 $ yarn add slick-carousel

  2. Open up main.js and main.css to add the entry points for the package. If you're using the Slick Carousel then your theme JS and CSS would look like:

     /* resources/assets/scripts/main.js */
 // Import external dependencies
 import 'jquery';
 import 'slick-carousel/slick/slick.min.js';

    Note: The path to Slick's JS doesn't need the node_modules directory at the beginning because Sage's Webpack config has the node_modules directory added to resolve.

     /* resources/assets/styles/main.scss */
 ...

 // Import Slick from node_modules
 @import "~slick-carousel/slick/slick.scss";
 @import "~slick-carousel/slick/slick-theme.scss";

 /* sage/assets/styles/common/_variables.scss */
 ...

 $slick-loader-path:     '~slick-carousel/slick/';
 $slick-font-path:       '#{$slick-loader-path}fonts/';

    Note: The ~ is used by the Sass loader for Webpack to tell Webpack that it's not a relative import. More information can be found in the sass-loader README.

  3. After running yarn build from the theme directory, your package will be built with your theme assets. The dist folder will contain a vendor directory that has any assets referenced from your packages.

Additional examples

If you'd like to share any examples you've created, any contributions to the Sage 9 docs on 3rd party packages) would be very appreciated!

js-cookie

js-cookie is a simple API for handling browser cookies.

  1. yarn add js-cookie

  2. Import the library & example usage:

    // @ resources/assets/scripts/routes/common.js
import Cookie from 'js-cookie';

export default {
 init() {
   // JavaScript to be fired on all pages
   const exampleCookie = Cookie.get('example');
 },
};

Lozad.js

Lozad.js is a pure JS lazy loader with no dependencies.

  1. yarn add lozad

  2. Import the library & example usage:

    // @ resources/assets/scripts/routes/common.js
import lozad from 'lozad';

export default {
 init() {
   // JavaScript to be fired on all pages

   // Lazy load images
   const observer = lozad('.lozad', {
     rootMargin: '250px 0px',
     threshold: 0.1,
   });
   observer.observe();
 },
};

Isotope

  1. yarn add isotope-layout imagesloaded

  2. Import the library & example usage:

    // @ resources/assets/scripts/main.js
// Import external dependencies
import 'jquery';
import 'isotope-layout/dist/isotope.pkgd.min';
import 'imagesloaded/imagesloaded.pkgd.min';

// @ resources/assets/scripts/about.js
const $grid = $('.about-team .row').imagesLoaded(function() {
 $grid.isotope({
   layoutMode: 'fitRows',
 });
});
