Theme troubleshooting

Never used a Node.js based build process before? New to some of the modern PHP techniques in the theme? Not familiar with ES6?

It might take a minute to get the hang of things, or even to just get things working in the first place . Once your development environment has been ironed out and you've figured out the kinks, you'll be cruising along and writing modern WordPress themes in no time.

Before you continue, make sure error reporting is enabled on your development environment (with WP_DEBUG).

Checking the Sage category on Roots Discourse might prove to be helpful. Chances are someone else has already run into the same issue that you are. Posting your own threads could also be helpful to other users that run into a similar problem. Please don't hesitate to reach out for help!

Node.js, npm, yarn related errors

Removing the node_modules directory and re-running yarn and trying the build process typically fixes most issues.

It's important to be using a newer version of Node.js, preferably installed by nvm. Completely remove your existing Node.js and install nvm as mentioned in chapter 3.

Still need help? See our npm Debugging Guidelines for support.

If you can't manage to get your builds working after troubleshooting and re-installing Node.js with nvm, please open a new thread on Roots Discourse.

Functions or hooks not working

The error you've probably ran into looks like:

Warning: call_user_func_array() expects parameter 1 to be a valid callback, function 'my_function_name' not found or invalid function name

This error means that you're in a namespaced file and you did not pass the namespace when referencing your function name.

// This will cause an error
add_filter('wp_nav_menu_objects', 'my_nav_menu_objects', 10, 2);

// This will work
add_filter('wp_nav_menu_objects', __NAMESPACE__ . '\\my_nav_menu_objects', 10, 2);

JavaScript won't work

ES6 might be throwing you off. The .eslintrc.js file can be configured to be less strict. If you're unable to get a JavaScript library working with the theme after reading about 3rd party packages in chapter 5, please open a new thread on Roots Discourse.
